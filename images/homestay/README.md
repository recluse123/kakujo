# Homestay 图片文件夹

## 使用方法

请将民宿相关的图片文件放在此文件夹中，文件命名格式如下：

- `homestay_1.jpg` (或 .jpeg, .png, .webp)
- `homestay_2.jpg`
- `homestay_3.jpg`
- ...以此类推

## 支持的图片格式

- .jpg
- .jpeg
- .png
- .webp

## 注意事项

1. 图片文件名必须严格按照 `homestay_数字.扩展名` 的格式
2. 数字从 1 开始，按顺序递增
3. 系统会自动检测并只显示实际存在的图片
4. 如果图片不存在，会自动跳过，不会显示错误

## 示例

```
images/homestay/
  ├── homestay_1.jpg
  ├── homestay_2.jpg
  ├── homestay_3.png
  └── homestay_4.webp
```
